# LeafletAI

A dynamic 3D flipbook viewer for InDesign/Illustrator exports.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

## Deployment

This project is configured for deployment on Vercel.

1. Push to GitHub.
2. Import project in Vercel.
3. Deploy.
